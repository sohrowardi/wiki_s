  • Avro

    sudo apt install ibus-avro

    Restart system or Log out - Log in

    Go to Settings > Region & Language (Settings > Keyboard in 22.04)

    In Input Sources click +

    Click Bangla or Bangla (Bangladesh)

    Finally select Bangla (Avro Phonetic)

    Now it should work just fine.
    If not make sure your
    Keyboard Input Method System is > IBus

    Common Problem

    Avro Stops writing Bangla. Try to restart ibus.

    ibus restart

  • Telegram

    Method 1. Install Telegram with APT Manager:

    sudo apt install telegram-desktop

    sudo apt-cache policy telegram-desktop

    sudo apt autoremove telegram-desktop --purge

    Method 2. Install Telegram with Snap (Snapcraft):

    sudo snap install telegram-desktop

    sudo systemctl restart gdm3

    sudo snap refresh

    sudo snap remove telegram-desktop

    Method 3. Install Telegram with Flatpak

    flatpak install flathub org.telegram.desktop

    sudo flatpak uninstall --delete-data org.telegram.desktop
